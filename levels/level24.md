## Solution
This was the fourth level in a row that was particularly easy, however, as is almost every case, I had to use the developer tools to solve it.

![Level 24 inspected](../images/level24_inspected.png)

From the inspected code nothing interesting can be seen, except for a particular comment that says 995674663, and since to solve this level it was necessary to search on the internet, I did the most obvious thing, that is look for that telephone number on google

![Level 24 research result](../images/level24_research.png)

As you can see the result that google gave me is a 20 second video in which there is a xylophone, so I tried to replace sound in the url with xylophone — and boom, level 24 completed!
